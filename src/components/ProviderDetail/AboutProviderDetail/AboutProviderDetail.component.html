About the map

This map will display the residential broadband footprint of up to three providers at one time. Use the search box to look for a provider's name and add the footprint to the map. Click on the "Add Provider" link to get additional search boxes to add additional footprints (up to three).

About the graphs

The graphs on the page provide more information about each provider's reported residential broadband service. The first graph shows the fraction of the population that each provider’s reported footprint covers. Below that, the next graph displays the fraction of each provider’s reported service availability, as a function of reported broadband speed. Subsequent graphs show the fraction of each provider's footprint that relies on each technology, as a function of speed.

Choose "Download" or "Upload" speed, at the top of the page to change whether the graphs display information about download speeds or upload speeds.

Broadband providers report data at the census block level, and may not offer service to every home in every block in which they report service. The calculations used to create the graphs treat every location as having service, and may therefore over-estimate broadband coverage, particularly in areas with large census blocks.

About provider names

Service providers that file data with the Commission identify themselves using several different names: provider name; DBA (doing business as) name; and holding company name (parent company). These names are self-reported; and providers take different approaches to the names they use for marketing and advertising their services.

The provider-name search bar uses a name meant to capture the parent companies (holding company name). However, in some instances, that name may not be familiar to end users. If you are unable to find the name of an ISP, please go to the Fixed Broadband Deployment: Provider Name Lookup to search among the different names submitted by the providers. The first column (Holding Company Final) includes the name you will search for on this site; the other columns contain other names associated with that first column that may be more familiar.
